Als Softwaretester erweitere die vorhandenen Testfälle um ausreichend Testdatenkonstellationen um eine hohe Testabdeckung zu erzielen. Hierfür wandel deine zuvor erstelle manuelle Testfallbeschreibung in BDD Cucumber Gherkin Szenarios jeweils innerhalb eines Features Files um, und liste je Testdatenkonstellation als Iteration in der unter dem Gherkin Szenario befindlichen Datentabelle auf.
